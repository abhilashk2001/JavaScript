## 1) Where and How to Run JavaScript

**In the Browser Console**
- Open DevTools → Console tab → type small snippets.
- Use `console.log()` to inspect values.

**In a Code Editor (like VS Code)**
- Save files with `.js` extension.
- Link the JS file to an HTML document.

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello</h1>
    <script src="fundamentals.js"></script>
  </body>
</html>
